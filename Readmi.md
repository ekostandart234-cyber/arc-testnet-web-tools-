# Arc Testnet Web Tools

**Красивый и удобный веб-инструмент для Arc Testnet**

Создан для удобства builders и Architects в [community.arc.io](https://community.arc.io/)

### ✨ Функции

- Подключение MetaMask
- Обычная отправка USDC
- **Batch Send** через Multicall3 (экономия газа)
- Deploy контрактов (Counter и другие)
- Просмотр информации о сети в реальном времени

### Как запустить локально

```bash
pip install -r requirements.txt
streamlit run app.py
